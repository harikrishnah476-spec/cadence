# Cadence // Adaptive Rhythm Mastery

Cadence is a high-performance, evidence-based typing mastery platform designed for professional workflow optimization. Unlike generic typing trainers, Cadence uses an adaptive behavioral engine to identify rhythm instability and fatigue, tailoring sessions to your specific professional cadence (Code, Terminal, Office, or Writer).

## Core Philosophy
- **Outcome-Focused**: Training that translates directly into real-world productivity.
- **Evidence-Based**: Every recommendation is grounded in measurable behavioral data.
- **Minimalist & Focused**: No gamified noise; just pure rhythmic performance.
- **Adaptive Intelligence**: Local predictive models that prevent performance collapse.

## Technical Stack
- **Framework**: Next.js 14 (App Router)
- **State Management**: Zustand (with Persistence)
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel (Optimized for Free Tier)

## Key Features
- **Progressive Disclosure**: Guided calibration phase for new users.
- **Skill Environments**: CodeFlow, TerminalFlow, OfficeFlow, and WriterFlow.
- **Predictive Analytics**: Real-time rhythm instability and fatigue forecasting.
- **Longitudinal Intelligence**: Cross-session behavioral profile tracking.
- **Magic Moments**: Visual reinforcement for peak flow states (Focus Lock).

## Getting Started
1. **Clone the repository**: `git clone <repo-url>`
2. **Install dependencies**: `npm install`
3. **Run locally**: `npm run dev`
4. **Build for production**: `npm run build`

## Architecture Overview
- `/src/store`: State management for typing logic and coaching profiles.
- `/src/core`: Behavioral performance analyzer and predictive models.
- `/src/components`: High-performance, focus-centric UI components.
- `/src/lib`: Lightweight analytics and utility functions.

## Deployment Instructions
1. **GitHub**: Push your code to a GitHub repository.
2. **Vercel**: Import the repository into Vercel.
3. **Configuration**: Use default Next.js settings. No environment variables required for core functionality.
4. **Deploy**: Vercel will automatically build and deploy your application.
